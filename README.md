 # Advanced AI-Powered Face Recognition using PCA & ANN

## 📌 Project Overview
This project implements a Face Recognition System using:
- PCA (Principal Component Analysis) for dimensionality reduction
- ANN (Artificial Neural Network) for classification

The system improves recognition speed and accuracy by extracting key facial features.

---

## 🧠 Workflow
1. Load dataset
2. Image preprocessing
3. PCA feature extraction
4. Train ANN classifier
5. Evaluate model accuracy

---
